# Telegram Bot для Fares Korea

Telegram бот для сбора информации от пользователей и предоставления персональных рекомендаций.

## Настройка на Railway

### 1. Переменные окружения

В настройках сервиса на Railway (вкладка **Variables**) необходимо добавить следующие переменные:

- `TELEGRAM_BOT_TOKEN` - токен вашего Telegram бота (обязательно)
  - Получить можно у [@BotFather](https://t.me/BotFather) в Telegram
  - Формат: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`

- `ADMIN_CHAT_ID` - ID чата администратора для получения уведомлений (опционально)
  - Можно узнать у [@userinfobot](https://t.me/userinfobot)

### 2. Деплой

После добавления переменных окружения:
1. Railway автоматически перезапустит сервис
2. Проверьте логи (вкладка **Logs**) на наличие ошибок
3. Убедитесь, что видите сообщение: `✅ Бот запущен и база данных готова.`

### 3. Проверка работы

Отправьте боту команду `/start` в Telegram для проверки работоспособности.

## Локальная разработка

1. Установите зависимости:
```bash
npm install
```

2. Создайте файл `.env` в корне проекта:
```
TELEGRAM_BOT_TOKEN=ваш_токен
ADMIN_CHAT_ID=ваш_chat_id
```

3. Запустите бота:
```bash
npm start
```

## Структура проекта

- `bot.js` - основной файл бота
- `database.js` - настройка базы данных SQLite
- `guides/` - папка с PDF гайдами для отправки пользователям
- `database.sqlite` - файл базы данных (создается автоматически)

## Технологии

- Node.js
- node-telegram-bot-api
- SQLite3
